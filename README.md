# Todo API 📝

A REST API built with Node.js and Express.

## Routes
- GET    /todos      → get all todos
- GET    /todos/:id  → get one todo
- POST   /todos      → create a todo
- PUT    /todos/:id  → update a todo
- DELETE /todos/:id  → delete a todo

## How to run
1. Clone this repo
2. Run npm install
3. Run node index.js
4. Server runs on port 3200
